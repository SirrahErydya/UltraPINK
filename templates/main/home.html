Welcome to Light coral!

Light coral is a Python-based web application to visualize self-organizing maps (SOMs) created with the PINK framework. PINK is a tool to generate parallelized invariant Kohonen maps structuring data according to their morphological features. Thus, a semi-automatic analysis of the data is possible.

Inspect map

Visit the map that is currently loaded. You will see a cluster of prototype images displaying all morphologies in the dataset.

You can choose between different options to analyse the dataset.

View loaded maps

See all maps that were saved to the database. You can choose a map for inspection. The chosen map will be remembered. Visit it over the Inspect SOM page

Create new map

Upload the binary files created by PINK to save your SOM within this tool. Afterwards you will be able to load and inspect the map.

View sources

View the sources for PINK and Light coral on GitHub.
